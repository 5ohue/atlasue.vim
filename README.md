### Installation

#### Vim

```vim
Plug '5ohue/atlasue.vim'
```

Then run `:PlugInstall`.

#### Neovim (lazy)

```lua
lazy.setup({
    {'5ohue/atlasue.vim'}
})
```

### Usage

```vim
colorscheme atlasue
```
